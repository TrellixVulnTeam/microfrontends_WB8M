Architechture of Demo application

  • Remote :

    Small Microfronted Application which are coupled together to build complete application. These Microfrontend application can run on seprate server and domain.

  • Host/Shell :

    The collection of micro-frontend remotes is housed inside another website, the host of the web application.

Webpack 5 provide Module fedration which help to wireup all microfrontend application in single page application.

Nx Library provide built-in support for building micro-frontend with webpack and Module Fedration

We have Create a small example of Microfronted. Consisting of a Host application i.e. Shell and two microfrontend application i.e. remote and angular-view
